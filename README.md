# RandomName

Este proyecto contiene una página HTML simple para generar o mostrar nombres aleatorios.

## Cómo usar

1. Abre el archivo `index.html` en tu navegador.
2. La página te permitirá ver el contenido y navegar de forma sencilla.

## Publicación

Puedes publicar este proyecto usando GitHub Pages:
- Sube los archivos a un repositorio en GitHub.
- Renombra el archivo principal a `index.html`.
- Activa GitHub Pages desde la configuración del repositorio.
